### 基本功能[Image/Video]
* 降噪 Denoising
* 超分辨率 Super resolution
* 去雾 Dehaze
* 涂抹及风格化 Force denoise
* 高帧率和慢动作 High FPS and Slow motion
### 硬件需求
> CPU模式 (CPU mode)
系统内存大于4GB
RAM > 4GB
> GPU模式 (GPU mode)
仅支持NVIDIA显卡
显存不小于2GB
Only support NVIDIA cards
VRAM should > 2GB
