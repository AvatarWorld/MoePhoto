# MoePhoto
MoePhoto Image Toolbox萌图工具箱  
一个开源的图像修复小软件（更新中...）
感兴趣的加群320785467讨论

### 基本功能
* 图像降噪
* 超分辨率
* 去雾
* 涂抹及风格化
### 硬件需求
> CPU模式 （不需要cuda）
系统内存大于4GB

> GPU模式
仅支持NVIDIA显卡
显存不小于2GB
